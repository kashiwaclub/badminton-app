# バドミントン 試合ローテーション App (バドローテ App)

バドミントンサークルの練習で、試合の組み合わせ（ダブルス）とお休みの割り振りを自動で行うWebアプリです。
iPhoneなどのスマホのホーム画面に追加して、普通のアプリのように使用できます。

## 📱 アプリの起動方法（スマホ推奨）
1. スマホのブラウザ（iPhoneならSafari）でアプリのURLを開きます。
2. 画面下部の「共有ボタン（四角から上矢印）」をタップします。
3. メニューから「ホーム画面に追加」を選択します。
4. ホーム画面にできたアイコンから起動すると、フルスクリーンで本物のアプリのように使用できます。

---

## 🏸 主な機能と使い方

アプリの画面は「試合表示」と「メンバー管理・回数」の2つのタブで構成されています。

### 1. 初期設定とメンバー管理（練習開始時・途中）
「メンバー管理・回数」タブをタップして開きます。
- **参加メンバーの設定:** 男子1〜20番（青）、女子1〜20番（赤）のボタンが並んでいます。今日参加しているメンバーの番号をタップして「オン（色付き）」にしてください。不参加のメンバーは「オフ（グレー・取り消し線）」にします。
- **途中参加・途中離脱:**
  遅れて来た人がいたら、その番号をタップしてオンにするだけで、次回の抽選から自動で追加されます。途中で帰る人がいればオフにするだけで自動除外されます。

### 2. 試合の抽選（コントロールパネル）
「試合表示」タブに戻り、画面上部で条件を設定します。
- **コート数:** 当日使用するコートの面数（1〜3面）を選択します。
- **時間設定:** 1試合の時間を分数で入力します（初期値は8分）。
- **試合抽選ボタン:** ボタンを押すと、現在の参加メンバーから自動で組み合わせを計算し、コートに割り振ります。

### 3. 自動抽選のルール（アルゴリズム）
運営の負担や不公平感をなくすため、以下のルールで自動計算されます。
- **2連続休みの防止:** 前回お休みだったメンバーは、次の試合で確実に最優先で選出されます。
- **お休み回数の平準化:** メンバー全員のお休み回数ができるだけ平等になるように、回数が多い人を優先してコートに入れます。
- **ミックスの成立（重要）:** 男女比の偏りによる「男子3・女子1」や「女子3・男子1」の組み合わせを完全に禁止しています。各コートは必ず「男4（男ダブ）」「女4（女ダブ）」「男2女2（ミックスダブ）」のいずれかになります。

### 4. 試合タイマー機能
画面右上のヘッダーにタイマーが常駐しています。
- 「試合抽選」ボタンを押すと、設定した時間で自動的にタイマーがスタートします。
- 手動で「ストップ」「スタート」を切り替えることも可能です。
- 時間が0になると、画面にアラートが表示されて試合終了を知らせます。

### 5. データのセット
- **試合数をリセット:** 練習の枠が変わる時などに、試合数を第0試合に戻せます。
- **回数をリセット:** 全員の過去のお休みカウントをすべて0回にリセットします。
